## NBAle - 猜NBA球星

本项目是一个猜NBA球员的网页游戏，玩家根据属性提示推理出今日答案。

### 📊 数据规模

- **球员总数：440人**
- 涵盖名宿、超巨、全明星、角色球员和部分新生代球员

### 🎮 难度模式

| 难度 | 人数 | 说明 |
|------|------|------|
| 🏆 新手模式 | 37人 | MVP传奇（最简单） |
| 🟢 简单模式 | 143人 | 总冠军球员 |
| ⚡ 普通模式 | 200人 | 全明星球员 |
| 🔥 困难模式 | 240人 | 角色球员（最难） |
| 👴 老球迷模式 | 440人 | 全部球员（最全面） |

### 🚀 本地运行

```bash
cd serve/src
python app.py
```

访问 http://localhost:9000/

### 📁 项目结构

```
├── public/          # 前端页面
├── serve/src/       # Python后端
├── functions/api/   # Cloudflare Pages后端
├── data/nba/        # NBA球员数据
└── web/             # Vue版本（备用）
```
